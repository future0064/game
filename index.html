Score: 0 | Lives: 3
Controls: ← → move, Z/Space jump, X rocket fly, C switch character (1/2), Q/W/E choose special, F use special, R restart
At game start: press C to choose character, then press ENTER/SPACE to start.
Character 1 (Original): choose one special → Q Speed / W Triple Jump buff / E Shield, press F to use. Character 2 (Girl): fixed special → F activates 10s Fly Mode (hold X to fly).
